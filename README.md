## My Portfolio Website

Website portofolio pribadi yang menampilkan profil, project, dan sistem admin untuk mengelola konten portofolio.

# admin panel
## fitur admin
1. login admin
2. CRUD data portofolio seperti pendidikan, skills, dan proyek yang pernah di kerjakan
3. upload gambar proyek

# halaman user
## fitur user
1. menampilakan proyek yang pernah di kerjakan
2. menampikan skills
3. menampilkan pendidikan
4. responsive design

# teknologi yang di gunakan
1. PHP native
2. HTML5
3. MYSQL
4. CSS3
5. laragon (local development)

# struktur folder
## admin/
    dashboard.php #halaman admin & crud data
    koneksi.php
    login.php
    logout.php
    uploads/

## My_Profile/
    index.php #halaman utama
    gambar/
